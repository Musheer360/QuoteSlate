About QuoteSlate

QuoteSlate is a powerful API that provides access to a vast collection of inspirational quotes. Whether you're building a motivational app, a writing tool, or just looking for daily inspiration, QuoteSlate has got you covered!

API Endpoints

Get a Random Quote

Retrieve a random quote from our collection.

GET /api/quotes/random

Get a Random Quote by Maximum Length

Retrieve a random quote with a maximum character length.

GET /api/quotes/random?maxLength=50

Get a Random Quote by Minimum Length

Retrieve a random quote with a minimum character length.

GET /api/quotes/random?minLength=100

Get a Random Quote by Length Range

Retrieve a random quote within a specific character length range.

GET /api/quotes/range?minLength=50&maxLength=150
