About QuoteSlate

QuoteSlate is a powerful API that provides access to a vast collection of inspirational quotes. With support for length filtering and tag-based searching, you can easily find the perfect quote for your needs. Whether you're building a motivational app, a writing tool, or just looking for daily inspiration, QuoteSlate has got you covered!

API Endpoint

Get a Random Quote

Retrieve a random quote from our collection. Optionally filter by length and/or tags.

GET /api/quotes/random

Get a Quote by Length Range

Retrieve a random quote within a specific character length range.

GET /api/quotes/random?minLength=50&maxLength=150

Get a Quote by Tags

Retrieve a random quote with specific tags (comma-separated).

GET /api/quotes/random?tags=life,wisdom

Get a Quote by Tags and Length

Retrieve a random quote with specific tags and length constraints.

GET /api/quotes/random?tags=wisdom&minLength=50&maxLength=150

Available Tags

The following tags can be used to filter quotes:

motivation inspiration life wisdom love success leadership happiness change perseverance mindfulness growth courage gratitude resilience friendship creativity humility forgiveness patience integrity self-reflection empathy purpose justice harmony knowledge hope anger fear general
