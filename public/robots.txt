# API Endpoints:
# - GET /api/quotes/random - Get random quotes with filters
# - GET /api/quotes - Paginated quotes with search/sort
# - GET /api/quotes/by-author/:author - Quotes by specific author
# - GET /api/quotes/by-tag/:tag - Quotes by specific tag
# - GET /api/authors - List all authors with quote counts
# - GET /api/tags - List all available tags
# - GET /health - API health status
